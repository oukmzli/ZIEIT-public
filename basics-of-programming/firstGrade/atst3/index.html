Текстовi процесори

Текстовий процесор - прикладне програмне забезпечення, призначене для редагування форматування тектової інформації у вигляді, максимально приблизному до кінцевого документа

Основні етапи роботи з текстовим документом:

  1. Створення
  2. Редагування
  3. Форматування
  4. Збереження та друк
